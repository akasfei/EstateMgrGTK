本文件夹包含了C语言课程设计->楼盘信息查询系统的：

1. 可执行文件: EstateMgrGtk.exe
2. 源代码: 
    main.c,
    core.h,
    core_io.h,
    core_EST.h,
    core_BDN.h,
    core_CEL.h,
    core_query.h
3. CodeBlocks项目文件: EstateMgrGTK.cbp
4. Gtk+工具包及Glean: Gtk+/
5. 技术文档: docs.txt
6. C语言程序设计报告: EstateMgrGtk_report.docx

本系统的Debug与Release编译结果可以在bin/debug与bin/release文件夹下找到。文件夹的根目录下也有一份Release编译结果的副本。

有关Gtk+的问题请参阅http://www.gtk.org/及http://developer.gnome.org/gtk/stable/
